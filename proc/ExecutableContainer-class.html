ExecutableContainer

ExecutableContainer class final

A virtual container for executables that can be viewed as a ProcessHost.

This class is useful for testing or emulating processes without actually running them. It allows you to set up a container with executables that return a Process when run, and produce a ProcessHost that responds to the executables in the container.

Example

Creating a container with an executable that echoes input on stdout:

import 'package:proc/proc.dart';

void main() async {
  final container = ExecutableContainer();
  container.setExecutable('echo', (start) async {
    late final ProcessController controller;
    controller = ProcessController(
      processId: 1234,
      onInput: (data) {
        controller.addStdoutBytes(data);
      },
    );
    return controller.process;
  });
}
