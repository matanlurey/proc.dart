ProcessHost

ProcessHost class abstract base

Runs processes and listens for signals on the host platform.

This class is an abstract base class that delegates to the host platform, or to a custom implementation.

To create a default process runner, use ProcessHost.new.

To create a custom process runner, extend this class and override the methods as needed.

Example

final host = ProcessHost();
final process = await host.start('echo', ['Hello, World!']);
