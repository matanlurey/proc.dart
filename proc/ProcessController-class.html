ProcessController

ProcessController class final

A controller with the process it controls.

This class can be used to create an instrumented Process instance, either for implementing a custom ProcessHost (i.e. that does not delegate to dart:io), or for testing.

The controller can be used to simulate a process by adding data to its stdout and stderr streams by methods such as addStdoutLine and addStderrLine, and then completing the process with complete.

The controller can also be used to simulate input to the process's stdin stream by providing a callback to onInput.

Example

final controller = ProcessController();
controller.addStdoutLine('Hello, World!');
controller.complete(ExitCode.success);
